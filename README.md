# MHIF — Meta Housing Intelligence Framework

**Meta Ads + constructoras LATAM**

MHIF es una metodología de inteligencia comercial para analizar, diagnosticar y optimizar campañas de Meta Ads en constructoras, con foco inicial en Colombia.

## Misión

Estandarizar la toma de decisiones en marketing inmobiliario mediante un sistema de conocimiento basado en experiencia, evidencia y aprendizaje continuo.

## Cómo usar este repositorio

1. Leer `MHIF_SPEC.md`.
2. Leer `MHIF_STYLE_GUIDE.md`.
3. Cargar las carpetas en Claude Project Knowledge.
4. Usar los prompts de `10_PROMPTS/`.
5. Actualizar casos reales en `08_CASES/`.

## Estructura

- `00_CORE/`: ADN metodológico.
- `01_RULES/`: reglas de decisión.
- `02_SIGNALS/`: métricas y señales.
- `03_PATTERNS/`: patrones diagnósticos.
- `04_CONCEPTS/`: conceptos base.
- `05_FRAMEWORKS/`: motores de análisis.
- `06_PLAYBOOKS/`: procedimientos por escenario.
- `07_BENCHMARKS/`: benchmarks vivos.
- `08_CASES/`: casos reales anonimizados.
- `10_PROMPTS/`: prompts operativos para Claude.
