# RULE_001 — Meta optimiza para el evento configurado

## Definición

Meta busca maximizar la probabilidad de completar el evento seleccionado.

Si el evento es Lead, Meta buscará Leads, no compradores.

## Implicación

Nunca juzgar el algoritmo por resultados que no puede observar, como visitas a sala, separaciones o escrituras, si esos eventos no están conectados al sistema.

## Error frecuente

Esperar compradores cuando solo se está optimizando por formularios.
